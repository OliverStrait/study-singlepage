# Simple Note-page

Produce little single-page note-application that can create new notes and show old ones

## Challenge goal

- Using only HTML, CSS and JavaScript
- note page that has capability to insert new one and observe old notes.
- Notes are stored in Browser [WebStorage](https://developer.mozilla.org/en-US/docs/Web/API/Web_Storage_API) 

## Status (Work in progress)

- [x] Static page layout prototype (html-DOM, css)
- [ ] Javascript and Localstorage API
    - Store data in json-string 

## Stored datatype

``` json
{
    id: 1772008395138, 
    t: "Note title",
    date: "2026-02-12",
    note: "Whole text in",
}
 
```
